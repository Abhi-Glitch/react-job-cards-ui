# React Job Cards UI

A simple React project that displays multiple job cards with company information such as role, salary, and location.
This project was built to practice React component structure and passing data between components using props.

## Features

- Reusable React card component
- Job information displayed inside cards
- Clean card UI layout
- Responsive layout
- Buttons for UI demonstration (non-functional)

## Technologies Used

- React
- JavaScript
- HTML
- CSS
- Vite

## What I Learned

- Creating reusable React components
- Passing data using props
- Understanding props drilling
- Structuring a small React application

## Future Improvements

- Add functionality to buttons
- Improve responsiveness further
- Add dynamic job data
